# From One Hot to LLM

A comprehensive introduction to language models and modern AI, starting from the basics.

## 📚 Course Overview

This course provides a hands-on introduction to language modeling and AI, designed to help you understand the fundamental concepts behind modern generative AI like Large Language Models (LLMs). Starting with classical statistical methods, you'll build a solid foundation for understanding how machines can learn to read and generate text.

## 📖 Course Contents

### [Chapter 0: N-gram Markov Models](https://github.com/guihcs/from-one-hot-to-llm/blob/main/chap0/markov.ipynb)

Located in `chap0/markov.ipynb`, this foundational chapter covers the basic concepts of language modeling.


## 🛠️ Setup & Installation

### Prerequisites
- Python 3.9 or higher
- UV package manager (recommended) or pip

### Installation

```bash
# Clone the repository
git clone https://github.com/guihcs/from-one-hot-to-llm.git
cd from-one-hot-to-llm

# Install dependencies using UV
uv sync

# Or using pip
pip install -r requirements.txt
```
